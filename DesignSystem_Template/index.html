Lena Losik Design System

Clean, minimalistic, and elegant.

This website provides documentation for the design system used to style my portfolio website. This design system was created to support and display my portfolio content and to help visually communicate my style, skilset and creativity.


This design system uses the Bootstrap 5 framework and divided into three sections:

Foundations

This section includes typography, colors and logo design.

Components

This section includes reusable components for buttons, cards and accordion.

Patterns

This section includes examples of layout pattern for homepage and about me page.
