Navigation Bar

Each page includes navigation bar. This provides easy access to desired page.

  • PROJECTS
  • ABOUT
  • CONTACTS

Accessibility Considerations

Provide hight contrast for accesability and clarity. Indicate current page with distinct color.


Usage

Navigation bar should be included on all pages.


Code example

      

         <nav class="navbar navbar-expand-lg bg-light"> 
              <div class="container-fluid"> 
               <a class="navbar-brand" href="#"> 
               <img src="../images/Logo_bw.png" width="60" class="d-inline-block align-top" alt="llosik logo">  </a> 
               <button class="navbar-toggler custom-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation"> 
               <span class="navbar-toggler-icon">   </span> 
               <div class="collapse navbar-collapse justify-content-end nav-bg" id="navbarNav"> 
               <ul class="navbar-nav"> 
                <li class="nav-item">
               <a class="nav-link active navtext" href="#">PROJECTS  </a> 
                </li> 
                <li class="nav-item"> 
                <a class="nav-link navtext" href="#">ABOUT</a>
